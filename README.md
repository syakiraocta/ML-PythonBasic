
# ML-PythonBasic 🤖

Repositori ini berisi kumpulan tugas dan catatan belajar mengenai dasar-dasar pemrograman Python.

📂 Struktur Folder
1. Fundamentals
Berisi 5 modul utama dalam format `.ipynb` yang disusun secara bertahap:
* **Python1 - Basic**: Variabel, tipe data, dan input/output dasar.
* **Python2 - Logic**: Pengkondisian `if-else` dan perulangan (*looping*).
* **Python3 - Data Structures**: Fungsi (`def`), *tuple*, dan *dictionary*.
* **Python4 - Modules**: Dasar pemisahan kode ke dalam modul.
* **Python5 - OOP**: Implementasi *Class*, *Inheritance*, dan *Encapsulation*.

### 2. Packages & Latihan Python
Bagian ini berfokus pada praktik modular programming:
* **Packages**: Contoh penggunaan modul `aritmatika.py` pada program utama.
* **LATIHAN PYTHON**: Praktik pembuatan struktur *package* menggunakan `__init__.py` untuk menghubungkan modul `alpha.py` dan `beta.py`.

### 3. File Latihan_OOP
Pada bagian **Python5 (OOP)**, ada beberapa penyesuaian *syntax* agar program lebih stabil, seperti:

* Penggunaan `super().__init__` untuk pewarisan atribut dari kelas induk.
* Penerapkan *private attribute* (`__`) untuk keamanan data.
* Menambahkan validasi pada perhitungan rata-rata agar tidak terjadi *error* jika data kosong.
